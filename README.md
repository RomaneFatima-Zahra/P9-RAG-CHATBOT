# Projet P9 - Chatbot RAG – Puls-Events

## Objectif du projet
Ce projet a été réalisé dans le cadre d’une mission freelance pour **Puls-Events**, entreprise technologique spécialisée dans la recommandation d’événements culturels.

L’objectif est de concevoir un **Proof of Concept (POC)** d’un **chatbot intelligent** capable de répondre aux questions des utilisateurs sur les événements culturels à venir, en s’appuyant sur un **système RAG (Retrieval-Augmented Generation)**.  
Le chatbot combine **recherche vectorielle** et **génération de texte** pour produire des réponses pertinentes à partir des données issues de l’API **Open Agenda**.
